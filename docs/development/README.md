## 🛠️ 技术栈

**前端**:

-   React 18 + Hooks
-   Vite (构建工具)
-   TailwindCSS + NextUI (UI框架)
-   Jotai (状态管理)
-   React Router (路由)
-   i18next (国际化)

**后端**:

-   Rust
-   Tauri (跨平台框架)
-   Tokio (异步运行时)
-   Serde (序列化)

**工具链**:

-   pnpm (包管理)
-   TypeScript (类型检查)
-   Prettier (代码格式化)
-   Tauri CLI (构建工具)
