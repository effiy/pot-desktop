## 💡 使用技巧

### ⚡ 效率提升

-   **快捷键**: 为常用功能设置快捷键，提升操作效率
-   **多引擎**: 同时启用多个翻译引擎，对比翻译质量
-   **历史记录**: 利用翻译历史快速重用之前的翻译
-   **自动化**: 启用自动复制、自动播放等功能

### 🎯 质量优化

-   **上下文**: 提供上下文信息提高翻译准确度
-   **专业领域**: 根据内容选择合适的专业翻译引擎
-   **后编辑**: 对翻译结果进行人工校对和优化
-   **反馈学习**: 通过使用反馈不断优化配置

### 🔧 故障预防

-   **定期备份**: 定期备份配置和翻译历史
-   **网络检查**: 确保网络连接稳定
-   **权限设置**: 正确设置系统权限
-   **版本更新**: 及时更新到最新版本
